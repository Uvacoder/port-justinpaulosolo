  This is a blog post about <strong>Tailwind CSS</strong>. <br />
  We'll learn how to set up and use Tailwind CSS.
</div>

### 1. Create Next.js project

```
// yarn
yarn create next-app my-project
cd my-project
```

```
// npm
npx create-next-app my-project
cd my-project
```

### 2. Install Tailwindcss

```
yarn add -D tailwindcss postcss autoprefixer
yarn tailwindcss init -p
```

```
// npm
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p
```

### 3. Modify tailwind.config.js

<SyntaxHighlighter language="javascript">
  {`module.exports = {
  content: [
    "./pages/**/_.{js,ts,jsx,tsx}",
    "./components/\*\*/_.{js,ts,jsx,tsx}",
  ],
  theme: {
  extend: {},
  },
  plugins: [],
}`}
</SyntaxHighlighter>

### 3. Add directives to global CSS

<SyntaxHighlighter language="css">
  {`@tailwind base;
@tailwind components;
@tailwind utilities;`}
</SyntaxHighlighter>
